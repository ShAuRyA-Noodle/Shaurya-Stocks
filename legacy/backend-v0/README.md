# Stock Signal Intelligence Platform

Explainable ML-powered stock Buy/Hold/Sell signal platform with realistic backtesting and SaaS delivery.

# Project Overview
- End-to-end ML-driven trading research system

# Architecture
- Time-series alpha
- Cross-sectional alpha
- Risk & regime filters
- Ensemble decision layer

# Methodology
- Walk-forward validation
- No lookahead bias
- Transaction costs included
- Turnover measured

# Results
- Cross-sectional outperforms TS
- Ensemble is more conservative
- Risk gating reduces drawdowns
